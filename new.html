What is HTML?

Hypertext Markup Language

"HTML (Hypertext Markup Language) is the set of markup symbols or codes inserted in a file intended for display on a World Wide Web browser page. The markup tells the Web browser how to display a Web page's words and images for the user."

HTML Creates the content of web page without styling, it defines the structure of the page and the text and images within it.

What is CSS?

Cascading Style Sheets

"Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language."

CSS is used to style the HTML and make the page look good, it is used for colours, backgrounds, layout, font, and much more.

Box model


Box Model

What is box model?

"The CSS Box Model. All HTML elements can be considered as boxes. In CSS, the term "box model" is used when talking about design and layout. The CSS box model is essentially a box that wraps around every HTML element. It consists of: margins, borders, padding, and the actual content."

Here is an example of box model in use in 4 steps


Hello

STEP 1: content
The content is the inner most box which is just as big as the content being held (100px in this case because of width:100px)

<p style=" background:red; width: 100px; ">Hello</p>

Hello

STEP 2: padding
The padding adds more room inside of the content box (now 200px width because 100px + 50px + 50px)

<p style=" background:red; width: 100px; padding: 50px;">Hello</p>

Hello

STEP 3: border
The border adds size to the edge of the content (now 240px)

<p style=" background:red; width: 100px; padding: 50px; border:20px solid black;">Hello</p>

Hello

STEP 4: margin
The margin is space outside the element itself; pushing away other elements and moving the element itself
(this doesnt add width to the element but takes up space so now the element has taken up 340px)

<p style=" background:red; width: 100px; padding: 50px; border:20px solid black; margin: 20px;">Hello</p>

All padding/margin can be done on any side (all sides, topbotom/leftright or top/right/bottom/left)

Hello

padding: 50px(top and bottom) 20px(left and right)

<p style="color: white; background:blue; width: 100px; padding: 50px 20px;">Hello</p>

Hello

padding: 50px(top) 10px(right) 100px(bottom) 0px(left)

<p style="color: white; background:blue; width: 100px; padding: 50px 10px 100px 0px;">Hello</p>

Hello

This can also be done by doing them individually

<p style="color: white; background:green; width: 100px; padding: 50px; border-bottom:20px solid yellow;">Hello</p>



Centering

There are 2 main ways to center.

text-align:center;
This is for when you want to center inline elements (such as text) You use text-align:center; on the containter

EXAMPLE

Centered Title

hello this is centered! (using text-align: center;)

This container isn't centered

text-align:center; will not center block elements like a container or wrapper!

<div style="margin-bottom: 50px; text-align:center">

    <h2>Centered Title</h2>
    <img src="https://upload.wikimedia.org/wikipedia/commons/thumb/8/85/Smiley.svg/2000px-Smiley.svg.png" alt="centered img" width="100px" height="auto"/>
    <p> hello this is centered! (using text-align: center;) </p>

    <div style="width:200px;height:200px;background:red;" class="container">
        <p> This container isn't centered </p>
    </div>

</div>

The other way to center is:

margin: 0 auto;

This is for centering block elements like the div above or a wrapper.
0 (this is the top and bottom margin, it can be any number like 20px it doens't have to be 0)
auto (this is the important bit, it makes the block element have auto margin on both sides)

EXAMPLE

Nice

This is a centered wrapper using margin: 0 auto!

<div class="wrapper" style="width:800px; margin: 0 auto;">

Now we can put everything in this wrapper to keep the content centered.

Colour?

There are 3 main ways to do colour

red

keywords:

still red

hex:
Hex colours huse a hexadecimal code to make the rgb colours
for example: #ffffff is white
the first two numbers are the R value, then the G value, then the B value.
so #ff0000 is red

<p style="color: #ff0000;"> still red </p>

red red red

The last way is RGB value
this is when you define the red green and blue values

slightly transparent red

The rgb value is useful as you can add an alpha value to choose the opacity.
